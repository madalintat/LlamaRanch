Hardware-aware

llama.cpp's --fit auto-sizes GPU layers and context to the memory you actually have. No flags to tune.

load

One-click serving

Models load on demand and unload when idle. One router process, switched in a click.

vision

Multimodal models are detected and paired with their projector automatically. Chat about images.

Hybrid

Big models too

Models larger than your VRAM run split across GPU and RAM - no manual offload math.

How it works

 LlamaRanch (tray)  ──spawn + flags──▶  llama-server  (llama.cpp)
                    ◀──── /health ────
                                          serves  127.0.0.1:2276
                                            /v1   OpenAI API
                                            /     built-in WebUI

LlamaRanch never embeds llama.cpp - it drives the prebuilt llama-server binary. Update llama.cpp on your own schedule and the ranch keeps running.

Bring any GGUF.

Drop .gguf files in your models folder, or grab them from the catalog. Anything on Hugging Face that runs in llama.cpp runs here.

Install

Package

npm run tauri build -- --bundles deb
sudo dpkg -i src-tauri/target/release/\
  bundle/deb/LlamaRanch_*.deb

Then launch LlamaRanch from your app menu. Toggle Start on login in Settings.

From source

git clone https://github.com/madalintat/LlamaRanch
cd LlamaRanch
npm install
npm run tauri build -- --no-bundle

Needs Rust, Node 18+, and a built llama-server from llama.cpp.
